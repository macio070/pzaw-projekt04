# INSTRUKCJE URUCHOMIENIA

1. W konsoli: 
   - npm install
  
2. Przed pierwszym uruchomieniem programu w katalogu głównym należy utworzyć plik `.env`:
   ```
   POPULATE_DB=1
   ```

3. W konsoli:
   - node index.js

4. Po uruchomieniu aplikacji plik `.env` należy usunąć

5. Na potrzeby doświadczenia deweloperskiego zostały utworzone 3 konta:
   - konto administratorskie:
      - login: admin
      - hasło: admin
   - konto użytkownika:
      - login: user1
      - hasło: 1234
   - konto użytkownika:
      - login: user2
      - hasło: qwerty
